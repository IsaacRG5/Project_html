#  🎸 Shattered Riffs - Website

Sitio web oficial de la banda ficticia **Shattered Riffs**, diseñado para mostrar eventos, información del grupo, contenido multimedia y un formulario de contacto.

---

## 📌 Descripción

Este proyecto es una página web estática desarrollada con **HTML y CSS**, enfocada en una banda de rock. Incluye secciones como:

- Hero (portada principal)
- Eventos próximos
- Información de la banda
- Galería multimedia
- Formulario de contacto

También cuenta con diseño responsive para adaptarse a dispositivos móviles.

---

## 🛠️ Tecnologías utilizadas

- HTML5  
- CSS3  
- JavaScript (básico - `main.js`)

## 🚀 Funcionalidades

- Navegación entre secciones con anclas
- Tabla de eventos con fechas y ubicaciones
- Diseño responsive (media queries)
- Galería de imágenes tipo grid
- Formulario de contacto
- Header fijo (sticky)

---

## 📱 Responsive Design

El sitio se adapta a pantallas pequeñas (≤ 768px):

- Menú reorganizado
- Imágenes más pequeñas
- Layout flexible
